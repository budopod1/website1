How to install Xampp

Got to https://www.apachefriends.org/index.html

Download xampp for what ever operating system you are using

Install Xampp with default settings


Change Apache config by http.conf find Document root and change to your web site (Careful about /s and \s)

Save the file and restart Apache if needed

Go to localhost in your webbrowser


How to get a domain name

Go to https://my.freenom.com

find domain name you like

Click check and select use your own DNS
